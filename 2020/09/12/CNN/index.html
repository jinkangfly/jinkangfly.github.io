CNN

基本概念
CNN基本结构：
{输入层卷积激活层池化层全连接分层\begin{cases}&输入层\\ &卷积\\ &激活层\\ &池化层\\ &全连接分层\\ \end{cases}⎩⎪⎪⎪⎪⎪⎪⎨⎪⎪⎪⎪⎪⎪⎧​​输入层卷积激活层池化层全连接分层​

卷积：平移不变模式

​ 定义一种提取局部的特征的方法，可有效相应特定局部模式

池化：下采样被检测物体不变模式

​ 可以直接对图像进行缩放

卷积核的个数 = 下一层数据的深度 = 下一层卷积层卷积核的深度

卷积核的个数 = 提取特征的数量，超参数，可以自己调节

多层卷积：一层卷积得到的特征往往是局部的层数越高，学到的特征就越全局化

卷积层：

输入：W_{1}(宽) x H_{1}(高) x D_{1}(深度)

超参数：

​ k个卷积核(the number of filters: K)

​ 卷积核空间分辨率 F x F (the dimension of filters: F)

​ 步长 s(stride: s)

​ padding: p
输出： W_{2} x H_{2}(高) x D_{2}(深度)
W2=W1+2P−FS+1W_{2} = \dfrac{W_{1}+2P-F}{S} +1W2​=SW1​+2P−F​+1, H2=H1+2P−FS+1H_{2} = \dfrac{H_{1}+2P-F}{S} +1H2​=SH1​+2P−F​+1, D2=KD_{2} = KD2​=K
参数：(F∗x∗F∗x∗D1+1)∗x∗K(F *x* F *x* D_{1} + 1) *x* K(F∗x∗F∗x∗D1​+1)∗x∗K
